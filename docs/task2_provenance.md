## Clarification of “mura” boundary file content
- `admin_demographics/boundaries/mura.gpkg` represents the municipality level (shi–cho–son): cities (市), towns (町), and villages (村) within Aomori and Akita.
- A villages-only subset is exported separately as `admin_demographics/boundaries/mura_only.gpkg` (units whose Japanese municipality name ends with 村).
